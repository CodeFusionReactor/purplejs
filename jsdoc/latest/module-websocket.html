websocket

WebSocket javascript functions.

Example
// Require websocket functions.
var websocket = require('/lib/websocket');

Methods

(static) send(id, message)

Send a message to the specified session id. The message can either be a string or a stream.

Parameters:
Name Type Description
id string

Session id to send the message to.
message string | *

Text (or stream) message to send.

(static) sendToGroup(group, message, excludeId)

Send a message to the specified session group. The message can either be a string or a stream. It can also have an optional excludeId to exclude a specific session id.

Parameters:
Name Type Description
group string

Session group to send the message to.
message string | *

Text (or stream) message to send.
excludeId string

Session id to exclude from message sending.
